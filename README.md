# Baby Calc H5

面向 4 岁儿童的加减法训练 H5，采用低压节奏与关卡激励。

## 当前功能

- 关卡训练：A 热身关（10加） -> B 主训练关（10减+20加） -> C 挑战关（20减）
- 过关规则：A/B 关需题数达标且正确率 >= 80% 才进入下一关，未达标直接结算
- 时间软上限：不中断当前题，达到软上限后本题做完再触发本关结束判定
- 云朵软计时：绿色/中间/红色状态，数字倒计时仅作为家长参考
- 小挑战精灵台词：在关卡切换、挑战插入、恢复模式时解释变化
- 简化帮助：每题最多 1 次帮助（自动拆分文字提示 + 二选一）
- 温和抗拒检测：无操作时长达到“每题时长”后先询问，连续触发才进入恢复题
- 奖励动画：A/B/C 奖励弹窗播放 `video/jiesuan_{num}.mp4`（`num=1..12`），可点击跳过
- 结算烟花：结算弹窗出现时自动播放
- 本地 7 天画像：用于 20 减挑战解锁判断（当日不回锁）
- 音效开关（默认关闭）与全屏按钮

## 快速运行

直接在浏览器打开 `index.html`。

## 资源准备（重要）

为避免法务风险，仓库不含人物图片，请在项目根目录准备：

- `img/left.png`（进度条左侧移动人物）
- `img/right.png`（进度条右侧固定人物）

目录示例：

```text
baby-calc/
  index.html
  styles.css
  app.js
  img/
    left.png
    right.png
```

## 主要文件

- `index.html`：页面结构与各类浮层锚点
- `styles.css`：视觉样式、动画与响应式布局
- `app.js`：状态机、出题、帮助、计时、奖励与本地持久化

## 行为说明

- C 挑战关（20减）默认锁定；满足近 7 天条件后解锁
- 解锁条件：最近 2 次 10减正确率 >= 80%，最近 1 次 20加正确率 >= 70%，且无明显抗拒
- 检测到疑似抗拒时先温和询问；连续触发才进入恢复题（10加、10加、10减）
- 奖励弹窗默认约 10 秒，可点“跳过动画”立即继续
- 若本地缺少 `video/jiesuan_*.mp4` 文件，奖励视频区域会自动隐藏，但流程会继续

## SRT 批量替换脚本

脚本路径：

```text
scripts/replace_srt_font_color.sh
```

用途：递归遍历目录中的 `.srt` 文件，将严格匹配的 `<字体颜色=` 替换为 `<font color=`。

执行示例：

```bash
./scripts/replace_srt_font_color.sh
./scripts/replace_srt_font_color.sh /path/to/srt-dir
```
