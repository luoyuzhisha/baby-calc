# Baby Calc H5

全AI生成的项目，给小朋友练习加减法的 H5 页面，支持：
- 10/20/50 以内加减法切换
- 5~60 秒倒计时（步进 5 秒）
- 10 题一轮练习与结果统计
- 进度条人物动画、全屏模式、平板适配

## 快速运行

直接在浏览器打开 `index.html` 即可运行。

## 资源准备（重要）

为了防止法务风险，本仓库不提供动画人物图片。
如果你是在新环境部署，请先准备图片资源：

1. 在项目根目录创建 `img` 目录（如果不存在）
2. 在 `img` 目录中放入以下文件：
   - `left.png`（进度条左侧移动人物）
   - `right.png`（进度条右侧固定人物）

路径示例：

```text
baby-calc/
  index.html
  styles.css
  app.js
  img/
    left.png
    right.png
```

## 主要文件

- `index.html`：页面结构
- `styles.css`：样式与响应式布局
- `app.js`：出题、倒计时、判题、统计与交互逻辑

## 说明

- 切换“10/20/50以内加减法”会重置本轮进度与计时。
- 倒计时下拉选择值在范围切换时会保留。
